# 📦 Inventory Management System

A modern, efficient Inventory Management System built with **ASP.NET Core MVC**. Designed to help businesses streamline inventory, suppliers, and purchasing workflows.

---

## ✨ Features

- 📋 Product Management (Add, Edit, Delete)
- 🛒 Supplier and Purchase Handling
- 📦 Real-time Inventory Tracking
- 🎨 Razor Pages with Bootstrap UI
- 🏛️ Clean MVC architecture

---

## 🚀 Technologies Used

- C# .NET 8 (or your targeted version)
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5

---

## 🛠️ Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Vaydelux/InventoryManagementSystem.git
```

### 2. Configure the Database
- Update the `appsettings.json` with your SQL Server connection string.
- Apply Migrations:
```bash
dotnet ef database update
```

### 3. Run the Application
```bash
dotnet run
```

Open your browser at:
```
https://localhost:5001
```
